# HoTroHocTap - Hệ thống tính điểm thông minh

## 📚 Giới thiệu
Hệ thống hỗ trợ sinh viên tính điểm học phần, quy đổi chứng chỉ, dự đoán đỗ đại học.

## 🚀 Tính năng
- Tính điểm học phần (3 thang điểm)
- Tính điểm qua môn
- Tính điểm tốt nghiệp
- Quy đổi chứng chỉ TOEIC/IELTS
- Dự đoán đỗ đại học (tỉ lệ %)
- Tham khảo điểm chuẩn các trường
- Dark mode
- Gửi góp ý qua Discord Webhook

## 📁 Cấu trúc
- `index.html` - Giao diện chính
- `style.css` - CSS
- `script.js` - JavaScript
- `data.js` - Database trường đại học
